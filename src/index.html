Overall Design
  1. A custom question per username before question list
  2. Access vault with answers of question list
  3. Create temp PIN to lock/unlock
  4. Back to step 1 if logged out or on new device
Decryption Process
  1. Combine the answers
  2. Get the hash of the answer-compound
  3. Get 1 part in the hash as auth password
  4. Download data with the auth password
  5. Load the data file
  6. Use the answer-compound as decryption password
  7. Decrypt the data file
Some Services
  • Proton Drive: [Pro]With pw-gated file public URL to access coz nothing is before the first vault. [Con]No API to automate and must do manual download.
  • Proton Pass or others: Vault services
  • MEGA, OneDrive, Dropbox: Pw-gated link is pay-to-use
  • G Drive: No pw-gated links for files
